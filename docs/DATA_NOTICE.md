# Data Notice

This repository contains synthetic demo data only.

The seed files in `demo-data/seed` were created for portfolio demonstration and are not exports, samples, subsets, screenshots, or transformed versions of production data.

## Synthetic Data Rules

- Use `DEMO-*` identifiers.
- Use arbitrary demo timestamps and numeric values.
- Avoid customer, facility, equipment, operator, production, lot, part, and defect identifiers from any real environment.
- Avoid real database dumps and operational logs.
